# Wallet Transaction Service

A backend wallet service handling deposits, withdrawals, and transfers with transaction safety. Built to simulate real fintech backend patterns — transaction atomicity, concurrency control, caching, and risk detection.

## Planned Features

- Wallet creation & management
- Deposit / Withdraw with validation
- Transfer between wallets (atomic)
- Transaction history & lookup
- Persistent storage with transaction safety
- API with Vert.x
- Redis caching
- Risk detection (daily limits, suspicious amount flagging, velocity checks)
